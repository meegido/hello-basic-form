Con este formulario básico quiero mostrar algunos de los elementos que ayudan a completar los objetivos de muchas webs y del usuario en la interacción de uno con el otro. De momento HTML y CSS, la base para experimentar dándole interactividad con JavaScript.

LABELS:

le dicen al usuario qué significa cada campo a completar.

Prueba a completar estos campos, con label y sin label, y envía el formulario.

INPUTS:

es uno de sus componentes básicos, permiten al usuario dar feedback.

Este tipo de botón circular solo permite seleccionar una opción.
¿Cómo se llama este input type?

Sin embargo, este cuadrado permite seleccionar más de una opción.
¿Cómo se llama este input type?

Y con este seleccionamos entre un listado la opción que deseamos.
La selección puede ser múltiple pero en este caso no lo es.
Si quisiéramos que fuera múltiple, ¿qué escribiríamos en input?

AYUDA:

mientras el usuario completa el formulario.
