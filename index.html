BASIC
FORM

Los formularios son una de las formas de interacción hombre-ordenador y continúa siendo una forma predominante en la interacción de los usuarios con la web. Con este formulario básico quiero mostrar algunos de los elementos que ayudan a completar los objetivos de la web y el usuario. De momento HTML y CSS, la base para experimentar dándole interactividad con JavaScript.

LABELS:

le dicen al usuario qué significa cada campo a completar.

Prueba a completar estos campos, con label y sin label, y envía el formulario.

INPUTS:

es uno de sus componentes básicos, permiten al usuario dar feedback.






AYUDA:

mientras el usuario completa el formulario.

Evitamos frustración y confusión usando mensajes con instrucciones o de error con un lenguaje sencillo y sin florituras.
